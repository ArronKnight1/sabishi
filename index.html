Sabishi Text RPG

A samurai themed Python text RPG game made on the Raspberry Pi.

Welcome!

Will you survive?

Sabishi is a fun RPG game set in feudal Japan. Battle enemys like ronin and samurai that have different attacks. Face bosses and encounter funny NPCs! The aim is to get the highest score you can!
